#vim #vim/copy

То же самое, что и при [выделении](Как%20выделить%20кусок%20кода.md), только вместо `v` используется `y`

Вместе со скобками, кавычками
```
ya "
```
Только содержимое
```
yi "
```

Блок
```
yi {
```
Блок со скобками, кавычками
```
ya {
```
Текстовый блок
```
ya p
```
