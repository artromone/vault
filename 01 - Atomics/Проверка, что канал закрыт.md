Проверка, что не "дефолтное значение пришло", а "канал закрыт"

```go
go func() { // Возведение в квадрат
	for {
		х, ok := <-naturals
		if !ok {
			break // Канал закрыт и опустошен
		}
		squares <- х * х
	}
	close(squares)
}()
```

Более лаконичный синтаксис:

```go
go func() { // Возведение в квадрат
	for x := range naturals {
		squares <- x * x
	}
	close(squares)
}()
```
