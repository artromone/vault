| **[[Методы HTTP]]** | CRUD                 |
| ------------------- | -------------------- |
| GET                 | Read                 |
| POST                | Create               |
| PUT                 | Update (замена)      |
| PATCH               | Update (модификация) |
| DELETE              | Delete               |
PUT – полностью заменить какой-то объект
PATCH –  изменить поля какого-то объекта
