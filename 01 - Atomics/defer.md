#prog_lang/golang 

Если defer несколько, они выполняются в порядке, обратном их объявлению

```go
defer ... // 1
defer ... // 2
```
```go
// defer 2
// defer 1
```

> Аргументы вычисляются на момент объявления этих функций (а не когда функция будет вызвана)
> **Переменные на момент инициализации**
