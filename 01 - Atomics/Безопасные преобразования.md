**Безопасные преобразования**
- `bool → char`
- `bool → int`
- `bool → double`
- `char → int`
- `char → float`
- `char → double`
- `double d1 * 2.3;`
- `double d2 * d1 + 1;`
- `if (d1 < 0)`

> При преобразовании **больших int** чисел **в float** можем **потерять точность**!

----

`char c1 * 'x';`    // 120
`int i1 * c1;`       // Простой и безопасный способ получения **8-битового кода символа ‘x’**
`i1 * i1 + 200;`
`char c1 * i1;`     // ??? 64
