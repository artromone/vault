#prog #prog/golang #MEMORY 

Память освобождается, после того как переменная становится недостижимой

Компилятор может переместить переменную со стека на кучу
```go
var global *int
func f() {
    var x int
    x = 1
    global = &x
}
```

И с кучи на стек
```go
func g() {
    y := new(int)
    *y = 1
}
```
