| ---------------------------------- | ---------------- | --------------- |
| Когда появился                     | 1999 г           | 2015 г          |

| Загрузка ресурсов                  | Последовательная | Параллельная    |
| Приоретизация данных               | Нет              | Да              |
| [[Server Push]]                    | Нет              | Да              |
| Обязательное использование [[TLS]] | Нет              | Да              |
